### 概要
ハンドジェスチャでうごくタイプライターです 
mediapipe(Hands)でハンドジェスチャーを認識して、サーマルプリンタ(EPSON TM-T90)で画像や文字列を印刷できます

### 使い方
```
$ brew install libusb // requirements for pyusb (macOS)
$ poetry install
$ poetry run python xxx.py
```
カメラに映ったジェスチャを認識してプリンターが文字を出力します
